## Getting Started

### `coral init`

Create a new task directory with a scaffolded packaged grader.

```bash
coral init <path> [--name NAME]
```

| Flag | Description |
|------|-------------|
| `path` | Path for the new task directory |
| `--name` | Task name (default: directory name) |

```bash
coral init my-task
coral init my-task --name "My Task"
```

Produces a self-contained layout: `task.yaml` wired to a packaged grader (`grader.entrypoint`), a baseline `seed/solution.py`, and a hatchling-built `grader/` package at `grader/src/<task>_grader/`. Run `coral validate <path>` next to bootstrap the grader venv and dry-run the grader against the seed.

### `coral validate`

Validate task structure and dry-run the grader against seed code.

```bash
coral validate <path>
```

| Flag | Description |
|------|-------------|
| `path` | Path to the task directory |

```bash
coral validate my-task
```

## Running Agents

### `coral prepare`

Create a timestamp run, shared state, repo clone, and per-agent worktrees
without launching agents.

```bash
coral prepare -c <task.yaml> [options]
```

```bash
coral prepare -c task.yaml
coral prepare -c task.yaml agents.count=4
coral validate --run-dir results/<task>/latest/.coral
```

### `coral start`

Launch autonomous agents from a prepared timestamp run. Auto-wraps in tmux by default.

```bash
coral start -c <run>/.coral/config.yaml [options]
```

| Flag | Description |
|------|-------------|
| `-c, --config` | Path to a prepared run's `.coral/config.yaml` (required) |
| `key=value` | Optional config overrides, e.g. `agents.count=4`, `agents.model=opus`, `run.ui=true`, `run.session=local` |

```bash
coral start -c results/<task>/latest/.coral/config.yaml
coral start -c results/<task>/latest/.coral/config.yaml run.ui=true
```

### `coral resume`

Resume agents from a previous run.

```bash
coral resume [--task TASK] [--run RUN] [options]
```

| Flag | Description |
|------|-------------|
| `--task` | Task name (auto-detected if omitted) |
| `--run` | Run ID (defaults to latest) |
| `-i, --instruction` | Additional instruction to inject into resumed agents |
| `--instruction-file` | Read additional resume instruction from a file |
| `key=value` | Runtime-safe overrides only, e.g. `agents.model=opus`, `run.verbose=true` |

```bash
coral resume
coral resume --task my-task agents.model=opus
```

### `coral stop`

Gracefully stop the CORAL manager and all agents.

```bash
coral stop [--task TASK] [--run RUN] [--all]
```

| Flag | Description |
|------|-------------|
| `--all` | Stop all active runs |
| `--task` | Task name |
| `--run` | Run ID |

### `coral status`

Show manager/agent status and top leaderboard entries.

```bash
coral status [--task TASK] [--run RUN]
```

## Inspecting Results

### `coral log`

List and search attempts. Default: top 20 sorted by score.

```bash
coral log [options]
```

| Flag | Description |
|------|-------------|
| `-n, --count N` | Number of results (default: 20) |
| `--recent` | Sort by time instead of score |
| `--agent ID` | Filter by agent ID |
| `--search QUERY` | Full-text search |
| `--task` | Task name |
| `--run` | Run ID |

```bash
coral log                     # Top 20 by score
coral log -n 5                # Top 5
coral log --recent            # Sort by time
coral log --agent agent-1     # Filter by agent
coral log --search "kernel"   # Full-text search
```

### `coral show`

Show full details and diff for a specific attempt.

```bash
coral show <hash> [--task TASK] [--run RUN]
```

| Flag | Description |
|------|-------------|
| `hash` | Commit hash or prefix |

```bash
coral show abc123
```

### `coral kb`

Query and maintain CORAL knowledge through a small index-first CLI.

```bash
coral kb index manual|external|practice [options]
coral kb read <id>
coral kb add external <source> --title "..." [--summary "..."]
coral kb remove <src-id>
coral kb note "short finding"
coral kb notebook [--agent agent-1] [--set plan.md]
coral kb archive <attempt-hash> --method-file method.md --reflection-file reflection.md
```

```bash
coral kb index manual
coral kb index practice --by score
coral kb read node-agent-1-0001
```

### `coral skills`

List skills or show details of a specific skill.

```bash
coral skills [--read NAME]
```

| Flag | Description |
|------|-------------|
| `-r, --read NAME` | Show details of a skill |

```bash
coral skills
coral skills --read optimizer
```

### `coral runs`

List all CORAL runs.

```bash
coral runs [options]
```

| Flag | Description |
|------|-------------|
| `-a, --all` | Include stopped runs |
| `-t, --task NAME` | Filter by task name |
| `-n, --count N` | Number of results (default: 20) |
| `-v, --verbose` | Show full paths |

```bash
coral runs                    # Active runs only
coral runs --all              # Include stopped
coral runs --task my-task     # Filter by task
```

## Dashboard

### `coral ui`

Start the CORAL web dashboard.

```bash
coral ui [options]
```

| Flag | Description |
|------|-------------|
| `--port PORT` | Port (default: 8420) |
| `--host HOST` | Host (default: 127.0.0.1) |
| `--task` | Task name |
| `--run` | Run ID |
| `--no-open` | Don't auto-open browser |

```bash
coral ui
coral ui --port 9000
```

## Agent Internals

These commands are primarily used by agents during their eval loop, but can also be run manually.

### `coral eval`

Stage all changes, commit, and run the grader.

```bash
coral eval -m <message> [--agent ID] [--workdir DIR]
```

| Flag | Description |
|------|-------------|
| `-m, --message` | Description of changes (required) |
| `--agent` | Agent ID (default: read from `.coral_agent_id`) |
| `--workdir` | Working directory (default: cwd) |

```bash
coral eval -m "Optimized inner loop with vectorization"
```

### `coral diff`

Show staged and unstaged changes.

```bash
coral diff [--workdir DIR]
```

### `coral revert`

Reset to HEAD~1, discarding the last commit and its changes.

```bash
coral revert [--workdir DIR]
```

### `coral checkout`

Reset the working tree to a previous attempt's commit.

```bash
coral checkout <hash> [--workdir DIR]
```

```bash
coral checkout abc123
```
